## Introduction 
Linear Search is one of the easiest searching algorithm.
The main aim of a searching algorithm is to search for a 
desired element in the whole array. Linear Search sequentially 
searches for an element in an array.

## Algorithm 
* In Linear Search, we start from the 0th index of the array,
we compare each element with the key (desired element).

* If the key is same as the element of array ,we return the index
of that element.

* If the key does not match any element then ,we return -1.

//Function for Linear Search.
void linearSearch(int a[],int size,int key)
{
  for(i=0;i<size;i++)    //traversing the array
  {
    //check whether key is present or not
    if(a[i]==key)       
    {
      //if found return i
      cout<<"Element found at "<<i<<endl;
      break;       
    }
  }
   // if not found
  if(i==size){
       cout<<"Not found"<<endl;
}
}

## Complexity Analysis

Linear search traverses the whole array while checking
key. The best case occurs when the key is present at the
first position itself, then only one comparison is made.
So the time complexity here is O(1).The worst case occurs 
when the element is not present in the whole array.So
the time complexity here is O(n).In linear search, 
no extra space is used to store anything. So the space complexity is `O(1)`.

* Time Complexity: `O(n)`
* Space Complexity: `O(1)`
